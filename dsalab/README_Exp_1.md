# Experiment 1: Revision of Built-In Data Structures

**Aim:** To analyze e-commerce data using Python's built-in data structures (Lists, Tuples, Dictionaries).

**Algorithm:**
1. Initialize `products`, `purchases`, and `searches` datasets.
2. Extract product names and find the max price.
3. Count purchases per user and sales per product.
4. Calculate revenues: price * quantity sold.
5. Identify the most searched product and best category.

**Output:** Printed summary of product analysis, user activity, total revenue (105,500), and search trends.
